# FrontWithAngular


```sh
ng generate environments

//----------------------- 
ng generate module features/suppliers --routing

// Create a file suppliers.routes.ts inside features/suppliers/
ng generate component features/suppliers/supplier-form
ng generate component features/suppliers/supplier-list
ng generate component features/suppliers/supplier-detail
ng generate component features/suppliers/supplier-add
ng generate component features/suppliers/supplier-edit

ng generate service _services/supplier
ng generate interface _models/supplier
ng generate interface _models/supplierFormData

//----------------------- 
ng generate module features/products --routing

// Create a file products.routes.ts inside features/products/ 
ng generate component features/products/product-list
ng generate component features/products/product-detail
ng generate component features/products/product-add
ng generate component features/products/product-edit
ng generate component features/products/product-form

ng generate service _services/product
ng generate interface _models/product 
ng generate interface _models/productFormData
```
